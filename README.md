# Анализ продаж компьютерных игр интернет-магазина «Стримчик»

## Описание проекта
Анализ исторических данных о продажах компьютерных игр до 2016 года. 
Цель — выявить закономерности успешности игр и сформировать рекомендации 
для планирования рекламных кампаний на 2017 год.

## Данные
- **Источник:** интернет-магазин «Стримчик»
- **Охват:** глобальные продажи (Северная Америка, Европа, Япония), до 2016 г.
- **Размер:** 16 715 игр, 11 признаков
- **Тип данных:** продажи по регионам, оценки критиков и пользователей, жанры, платформы, рейтинг ESRB

## Ход работы
- Предобработка данных: обработка пропусков, приведение типов, удаление дубликатов
- EDA: анализ продаж по платформам, жанрам и регионам
- Анализ корреляции оценок критиков и пользователей с продажами
- Портрет пользователя по регионам
- Проверка статистических гипотез (t-тест)

## Ключевые выводы
- **Актуальные платформы:** PS4, XOne и 3DS — лидеры продаж 2015–2016 гг.; именно на них стоит сфокусировать рекламный бюджет на 2017 год
- **Жанры:** Shooter и Action генерируют наибольшую выручку глобально; для японского рынка приоритет — Role-Playing
- **Оценки критиков:** умеренная корреляция с продажами на PS4 (r=0.39) и XOne (r=0.42) — высокий critic score является сигналом для продвижения игры
- **Региональные различия:** Северная Америка и Европа имеют схожие предпочтения; Япония — отдельный рынок с другими топ-платформами (3DS) и жанрами (Action, Role-Playing)
- **Рейтинг ESRB:** влияет на продажи в США и Европе (лидер — рейтинг M), но не в Японии
